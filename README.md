# ❄️ FRACTA Axis | Vault & Auth

**Fracta Axis** es la primera implementación comercial de la **Matemática Fractal del Universo (MFSU)**. Una plataforma diseñada para la seguridad total mediante algoritmos de encriptación inspirados en el crecimiento natural y el caos controlado.

## 🚀 Características
- **FRACTA Vault:** Encriptación de archivos (PDF, Imágenes, Office) usando el factor $\delta_F = 0.921$.
- **FRACTA Auth:** Generador de códigos 2FA dinámicos basados en evolución fractal.
- **Zero-Knowledge:** Tus archivos se procesan localmente; Fracta no guarda tus contraseñas ni tus datos.

## 🛠️ Instalación y Uso Local
1. Clona el repositorio:
   `git clone https://github.com`
2. Instala las dependencias:
   `pip install -r requirements.txt`
3. Corre la aplicación:
   `streamlit run fracta_vault.py`

## 🧪 El Algoritmo (MFSU)
Nuestra tecnología utiliza una semilla fractal que evoluciona con el tiempo y el ruido del sistema, haciendo que la clave de encriptación sea prácticamente imposible de predecir mediante métodos convencionales.
